10) SISTEMA DE OVERRIDES
════════════════════════════════════════════════════════════════════════════════════════════════

    ┌──────────────────────────────────────────────────────────────────────────────────────┐
    │ Objetivo: permitir flexibilidad sin romper la semántica del perfil                  │
    └──────────────────────────────────────────────────────────────────────────────────────┘

                     ┌──────────────────────────────────────────────┐
                     │ Estado por campo                            │
                     ├──────────────────────────────────────────────┤
                     │ allowed    -> se puede sobrescribir         │
                     │ restricted -> con limitaciones / validación │
                     │ forbidden  -> no se puede sobrescribir      │
                     └──────────────────────────────────────────────┘

    Ejemplo:
    ┌──────────────────────────────────────────────────────────────────────────────────────┐
    │ Perfil podcast                                                                      │
    │ - format.type      = audio      -> forbidden                                        │
    │ - format.extension = mp3        -> forbidden                                        │
    │ - preset_base      = base audio -> forbidden                                        │
    │ - quality          -> allowed                                                       │
    │ - max_items        -> allowed                                                       │
    └──────────────────────────────────────────────────────────────────────────────────────┘

    ┌────────────────────────────────────────────┐
